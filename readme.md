Knots 3D

To run this bot type in command line "python knots3d/manage.py bot". \
To get access to admin panel type in command line "python knots3d/manage.py runserver". It'll be
runned localy at "http://localhost:8000/admin". Password: admin , username: admin.

All telegram bot logic located in knots3d/bot/management/commands/bot.py. \
All Django setting in knots3d/knots3d/settings.py. \
All admin configs in knots3d/bot/admin.py. \
Database models in knots3d/bot/models.py.
